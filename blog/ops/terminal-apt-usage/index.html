Ubuntu apt 命令小结

安装

添加PPA源

PPA ,表示 Personal Package Archives，也就是个人软件包集。有很多软件因为种种原因，不能进入官方的 Ubuntu 软件仓库。 为了方便 Ubuntu 用户使用，launchpad.net 提供了 ppa，允许用户建立自己的软件仓库， 自由的上传软件。

#到[launchpad.net](https://launchpad.net/+search)搜索到ppa:user/ppa-name之后
root> add-apt-repository ppa:ubuntu-wine/ppa
sudo add-apt-repository ppa:openjdk-r/ppa
# 删除 ppa 源
root> apt-get install ppa-purge
root> ppa-purge ppa:ubuntu-wine/ppa

换源

ubuntu的源存放在/etc/apt/source.list,在换源之前我们要记得对之前的源进行备份，这样的好处是，万一你不行，你还可以复原回去。

root> cp /etc/apt/sources.list /etc/apt/sources.list.bak
root> vim /etc/apt/sources/list

安装

#搜索包
> apt-cache search {{package}}

#获取包的相关信息，如说明、大小、版本等
> apt-cache show {{package}}

#安装包
> sudo apt-get install {{package}} 

#重新安装包
> sudo apt-get install {{package}} --reinstall

查找已安装软件

> dpkg --list | grep jdk

更新软件

在修改/etc/apt/sources.list或者/etc/apt/preferences之后运行该命令，此外您需要定期运行apt-get update命令以确保您的软件包列表是最新的

#更新源
> sudo apt-get update 

#更新已安装的包
> sudo apt-get upgrade

卸载

删除不用的老旧内核

# 查看当前Ubuntu系统使用的内核
> uname -r

# 查看所有内核
> dpkg --get-selections |grep linux

# 删除
> sudo apt-get remove linux-image-xxxx

卸载软件

#卸载已安装的软件包
> apt-get remove {{packagename}}

#卸载一个已安装的软件包（删除配置文件）
> apt-get --purge remove {{packagename}}

#有些软件很难卸载，而且还阻止了别的软件的应用
> dpkg --force-all --purge {{packagename}}

清除缓存

#清除已经卸载的软件包的.deb文件
> sudo apt-get autoclean 

#将已安装软件包的.deb文件一并删除
> sudo apt-get clean 

# 删除系统不再使用的孤立软件
> sudo apt-get autoremove

举例

卸载 MySQL

#查看MySQL的依赖项
> dpkg --list|grep mysql

# 卸载
> sudo apt remove mysql-common
> sudo apt autoremove --purge mysql-server

# 清除残留数据
> dpkg -l|grep ^rc|awk '{print$2}'|sudo xargs dpkg -P

# 验证
> dpkg --list|grep mysql
