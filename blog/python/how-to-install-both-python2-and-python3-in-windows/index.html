  1. HostenWang

    Guest Author

概述|Overview

个人使用是 Python3, 个别行业软件只支持 #Python 2.7 ,结果就是安装 Python 2.7 后，本来在 Python 3 下能正常使用的脚本无法运行。

记录两个环境共存的配置

环境|Environment

名称 版本 备注
Windows 10 home x64
Python2 2.7
Python3 3.10

py 方式

脚本调用

手动指定

py -2 helloworld.py
py -3 helloworld.py

在脚本内第一行指定版本, 然后执行 py helloworld.py 即可

#! python3

pip

py -2 -m pip install watermark-painter
py -3 -m pip install watermark-painter
