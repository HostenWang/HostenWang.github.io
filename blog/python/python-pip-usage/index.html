概述

python pip 常用命令小结

安装卸载

安装时临时换源

配置

#清华源
pip install numpy -i https://pypi.tuna.tsinghua.edu.cn/simple
# 阿里源
pip install numpy -i https://mirrors.aliyun.com/pypi/simple/
# 腾讯源
pip install numpy -i http://mirrors.cloud.tencent.com/pypi/simple
# 豆瓣源
pip install numpy -i http://pypi.douban.com/simple/

永久换源

配置

# 清华源
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
# 阿里源
pip config set global.index-url https://mirrors.aliyun.com/pypi/simple/
# 腾讯源
pip config set global.index-url http://mirrors.cloud.tencent.com/pypi/simple
# 豆瓣源
pip config set global.index-url http://pypi.douban.com/simple/

# 换回默认源
pip config unset global.index-url

导出依赖

pip freeze > requirements.txt

卸载

单个卸载

pip uninstall requests

批量卸载

# If we want to remove all at once then
pip uninstall -r requirements.txt
pip uninstall -r requirements.txt -y

虚拟环境

virtualenv 使用

# 创建一个名字为envname的虚拟环境
virtualenv envname   
# 查看当前目录可以知道一个envname的文件已经被创建
dir     

# 如果安装了多个python版本，如py2和py3，需要指定使用哪个创建虚拟环境
virtualenv -p python2 envname

Q&A

The script automat-visualize.exe is not on PATH.

> echo 'export PATH=/usr/local/python3/bin:$PATH' >>~/.bashrc

> source ~/.bashrc
