Windows 10 home x64

报错

ORA-28002 The Password Will Expire in 7 Days Errors
ORA-28002: 7天之后口令将过期的解决方法

解决

打开 sqlplus 登录 sys/system 管理用户更改

-- 登录 sys/system 管理用户更改
SQL> sqlplus / as sysdba 

-- 查看用户密码默认管理方式
SQL> select profile from dba_users where username=users;

-- 查看数据库默认的密码管理方式有效期
SQL> SELECT * FROM dba_profiles WHERE profile='DEFAULT' AND resource_name='PASSWORD_LIFE_TIME';

-- 修改用户密码并设置数据库默认密码为永久有效)
SQL> alter user users identified by xxx;

-- 设置不限日期
SQL> ALTER PROFILE DEFAULT LIMIT PASSWORD_LIFE_TIME UNLIMITED ;
