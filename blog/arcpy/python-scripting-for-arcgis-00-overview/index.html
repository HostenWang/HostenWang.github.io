概述|Overview

ArcGIS 9 引入了脚本处理技术，并支持多种脚本语言，包括 Python、VBScritp 等.

由于 AreGIS 是基于组件对象模型（COM）构建起来的，又因为脚本语言是面向对象的，所以脚本语言可以访问 ArcGIS 中所有获得许可的函数，也包括所有的扩展模块。因此，脚本语言才可以高效地实现任务自动化，受到了程序员的青味。

目前，Python 已经成为 GIS 专业人员的一个基本工具。他们使用 Python 来扩展 ArcGIS 的功能并实现任务的自动化。
