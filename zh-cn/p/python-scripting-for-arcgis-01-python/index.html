概述|Overview

Python 是一门简单又强大的主流编程语言。由 Guido van Rossum 于1989年创建. Python 的类库非常丰富，这使得 Python 几乎无所不能，网站开发、软件开发、大数据分析、网络爬虫、机器学习等都不在话下。

Python 语言起源于 1989 年末，当时 CWI（荷兰国家数学与计算机科学研究中心）的研究员 Guido van Rossum 需要一种高级脚本编程语言，为其研究小组的 Amoeba 分布式操作系统执行管理 任务。为创建新语言，他从高级数学语言 ABC（ALL BASIC CODE）中汲取了大量语法，并从系 统编程语言 Modula-3 中借鉴了错误处理机制。他把这种新的语言命名为 Python（蟒蛇），Python 来 源于 BBC 当时正在热播的喜剧连续剧 Monty Python’s Flying Circus ，他希望这个新的叫作 Python 的语言能符合他的理想:创造一种处于 C 和 Shell 之间，且功能全面、易学易用、可拓展的语言。

在 GIS 领域, Python 可以用来控制 ArcGIS 工具和地图, 用于生成地理数据和地图, 基于优秀的 arcpy 模块可以极大的加速 ArcGIS 操作

Python的特点

  1. 简单易学
  2. 免费且开源
  3. 跨平台

ArcGIS 和 Python

ArcGIS for Desktop 支持使用脚本语言实现任务自动化.Python是ArcGIS首选的脚本语言

Python 脚本可以像常见的地理处理工具一样，变成一个脚本工具，直接整合到 ArcGIS 中。

ArcGIS 10 安装后自带一个 Python 2.7
