Windows 系统下 Oracle 的安装和配置|Oracle Installation and Configuration on Windows

Jun 04, 2018
阅读时长: 1 分钟

概述

本文介绍如何在 Windows 系统下安装 Oracle 11g 数据库

准备

安装需要 5G 左右的磁盘空间（Oracle 主目录 3GB，数据文件 2GB）

安装

Oracle 11g下载完成后，解压到一个空的文件夹，然后双击 setup.exe 文件，就可以开始安装Oracle数据库了

  • 数据库版本 选择企业版
  • 字符集 选择 Unicode(AL32UTF8)
  • 全局数据库名 即是数据库连接时的实例名，这里我设置为 orcl

一路确认,默认即可

口令管理

到了口令管理那里留意一下

单击【口令管理】按钮设置不同用户的密码，这里我们将 SYS，SYSTEM 和 HR 用户解锁并设置相应的密码 ，完成后，点击【确定】。

其他确认即可

验证

  1. 打开 Sql Plus

  2. 在命令提示符中输入用户和登录密码，如提示连接成功，证明我们的Oracle11g安装完

出错和解决

  1. 安装过程中可以会出现“由于以下错误,Enterprise Manager 配置失败 -实例化 EM 配置文件时出错”.请参考 Oracle - Windows下Oracle 11g数据库详细安装、设置教程 解决

  2. tms


打开注册表看看：HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services 下，找到OracleMTSRecoveryService及OracleRemExecService，查看ImagePath对应路径。

修改ImagePath路径为xxxxxxx\dbhome_1\bin\xxxxxxxxx即可【修改为你本地真实路径即可】，错误原因为你上次未卸载干净，实际路径错误
