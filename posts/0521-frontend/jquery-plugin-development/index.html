简单记录 jquery 插件开发

学会使用 jQuery 并不难，因为它简单易学，且插件丰富。编写一个属于自己的插件是个不错的选择。

jQuery 支持的拓展方式

jQuery 插件开发方式主要有三种：

  1. 通过$.extend()来扩展jQuery
  2. 通过$.fn 向jQuery添加新的方法
  3. 通过$.widget()应用jQuery UI的部件工厂方式创建
