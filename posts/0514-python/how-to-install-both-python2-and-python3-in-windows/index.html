如何在 Windows 上同时安装 Python2 和 Python3

介绍python2和python3的主要区别和如何在 Windows 上同时安装 Python2 和 Python3

Sep 25, 2017
阅读时长: 1 分钟

概述|Overview

个人使用是 Python3, 个别行业软件只支持 #Python 2.7 ,结果就是安装 Python 2.7 后，本来在 Python 3 下能正常使用的脚本无法运行。

Python 2 和 Python 3 主要差别

Unicode

Python 2有基于ASCII的str()类型，其可通过单独的unicode()函数转成unicode类型，但没有byte类型。

而在Python 3中，终于有了Unicode（utf-8）字符串，以及两个字节类：bytes和bytearrays。

print

Python 2中的print语句被Python 3中的print()函数取代，这意味着在Python 3中必须用括号将需要输出的对象括起来。

For循环变量与全局命名空间泄漏

在Python 3.x中，for循环中的变量不再会泄漏到全局命名空间中了！

触发异常

Python 2支持新旧两种异常触发语法，而Python 3只接受带括号的的语法（不然会触发SyntaxError）

异常处理

Python 3中的异常处理也发生了一点变化。在Python 3中必须使用“as”关键字
