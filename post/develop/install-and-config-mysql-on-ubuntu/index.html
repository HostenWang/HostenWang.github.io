Ubuntu 下 MySQL 的安装和配置|Installation and Configuration of MySQL in Ubuntu

Posted by HostenWang on Saturday, April 2, 2016

1 概述|Overview

Linux 下 MySQL 的安装和配置

2 Environment

ubuntu 14.04

MySQL 5.5

3 MySQL

apt-get update
apt-get install MySQL-server

# create user
MySQL -uroot -p
create user 'test'@'%' identified by 'test'

## database access
CREATE DATABASE `test` DEFAULT CHARACTER SET utf8 COLLATE utf8_general_ci
grant all on *.* to 'test'@'%';

# remote access
## 1
grant all privileges on *.* to 'test'@'%' with grant option;
flush privileges;
## 2
vim /etc/my.conf

# restart
service MySQLd restart

# verify
MySQL -h192.168.1.0 -utest -p

3.1 Firewall

# only for test
ufw disable

# for prod
iptables -A INPUT -p tcp -m tcp --dport 3306 -j ACCEPT
sudo iptables -nvL
