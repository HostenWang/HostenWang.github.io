Gitea 使用总结|Gitea Usage

Posted by HostenWang on Sunday, July 4, 2021

1 概述|Overview

2 环境信息

Ubuntu 20

Gitea 1.13.2

3 准备

升级系统 && 安装软件

apt update && apt full-upgrade -y
apt install git wget nginx ufw -y

3.1 暂时关闭防火墙

ufw disable

3.2 创建用户

useradd --home-dir /var/lib/gitea  --create-home --no-user-group --shell /bin/bash git

4 安装

4.1 建立安装目录

mkdir /opt/gitea/
wget https://dl.gitea.io/gitea/1.13.2/gitea-1.13.2-linux-amd64 -O /opt/gitea/gitea
chown root:root /opt/gitea/gitea
chmod 755 /opt/gitea/gitea

4.2 配置安装 Gitea

su -l git
/opt/gitea/gitea --work-path /var/lib/gitea --custom-path /var/lib/gitea web --config /var/lib/gitea/app.ini

5 配置

5.1 配置

访问 http://ip:3000,配置安装选项

5.2 开机启动

基于 supervisor 的开机启动和自动重启

[program:gitea]
directory=/opt/gitea/
command=/opt/gitea/gitea web
autostart=true
autorestart=true
startsecs=10
stdout_logfile=/opt/gitea/log/stdout.log
stdout_logfile_maxbytes=1MB
stdout_logfile_backups=10
stdout_capture_maxbytes=1MB
stderr_logfile=/opt/gitea/log/stderr.log
stderr_logfile_maxbytes=1MB
stderr_logfile_backups=10
stderr_capture_maxbytes=1MB
user = gitea
environment = HOME="/home/gitea", USER="gitea"

6 完成

ufw default deny
ufw allow 22/tcp
ufw allow 3000/tcp
ufw allow 80,443/tcp
ufw enable
