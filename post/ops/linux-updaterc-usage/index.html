Linux updaterc 使用总结|Linux updaterc Usage

常用命令和示例

Posted by HostenWang on Monday, December 11, 2017

1 概述|Overview

Ubuntu 18 不再使用 initd 管理系统，而是使用systemd

我们使用 update-rc.d，可以添加和删除服务，并将它们添加到 Ubuntu/``Debian` 启动脚本中，实现开启自启动效果。

2 环境|Environment

名称 版本 备注
Ubuntu 18 LTS

3 新增开机启动项

$ cd /etc/init.d/
$ vim serve.sh

#!/bin/bash
# command content
exit 0

$ chmod +x serve.sh

4 设置

# 90表明一个优先级，越高表示执行的越晚 （优先级范围：0~90）
$ update-rc.d serve.sh defaults 90

5 验证

# 查看全部服务列表
$ service --status-all

# 单独查看
$ service serve status

6 启动和停止

$ service serve start
$ service serve stop
$ service serve restart

7 移除

$ update-rc.d -f serve.sh remove
