Apache NiFi Error OutOfMemoryError unable to create new native thread

NiFi 错误排查系列

Posted by HostenWang on Saturday, January 18, 2020

1 概述|Overview

Apache NiFi 是一个易于使用、功能强大而且可靠的数据处理和分发系统。

2 错误|Error

java.lang.OutOfMemoryError: unable to create new native thread

3 排查

根据报错提示,怀疑是服务器资源限制导致

> ulimit -a

data seg size           (kbytes, -d) unlimited

file size               (blocks, -f) unlimited
pending signals                 (-i) 765590

max memory size         (kbytes, -m) 33554432
open files                      (-n) 65536
pipe size            (512 bytes, -p) 8
POSIX message queues     (bytes, -q) 819200
real-time priority              (-r) 0
stack size              (kbytes, -s) 8192
cpu time               (seconds, -t) unlimited
max user processes              (-u) 20000
virtual memory          (kbytes, -v) unlimited
file locks                      (-x) unlimited

4 解决

解决方案,任选其一

  1. 调大资源限额
  2. 使用 root 账户启动 nifi
