GeoServer 系列

Posted by HostenWang on Thursday, April 21, 2022

1 概述|Overview

GeoServer 是 OGC Web 的开源实现.

PostGIS shapefile 工具将 shapefile 数据从二进制转换为一系列的 SQL 命令，然后在数据库中运行以加载数据，从而使 shapefile 数据在 PostGIS 中可用

2 shapfile 包含的内容

类型 介绍 是否必选
.shp 存储地理要素的几何信息 必选
.shx 存储要素几何图形的索引信息 必选
.dbf 存储地理要素的属性信息（非几何信息） 必选
.prj 存储空间参考信息，即地理坐标系统信息和投影坐标系统信息。使用well-known文本格式进行描述 可选

3 基础概念

功能 参数
SRID(Spatial Reference IDentifier, 空间参考标识符) 它定义了我们数据的地理坐标系统和投影的所有参数
spatial_ref_sys 表 用于定义数据库已知的所有空间参考系统
Coordinate Reference Systems Native SRS
Declared SRS
SRS handling
Bounding Boxes Native Bounding Box
Lat/Lon Bounding Box
Isolated Workspace
