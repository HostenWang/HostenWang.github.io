在Windows安装PostGIS|Install PostGIS on Windows

Posted by HostenWang on Tuesday, September 7, 2021

1 概述|Overview

PostGIS是一个对象关系型数据库系统PostgreSQL的扩展，它把GIS（地理信息系统）类型对象存储在数据库中。

本文介绍 #PostGIS 基本安装

2 环境信息|Environment

名称 版本 备注
Windows 10 Home x64
postgresql 9.5.23
postgis 2.3

3 安装|Installation

分别下载对应安装包即可

4 配置|Configuration

打开 psql

> select postgis_full_version();
错误:  函数 postgis_full_version() 不存在
第1行select postgis_full_version();
            ^
提示:  没有匹配指定名称和参数类型的函数. 您也许需要增加明确的类型转换.

> create extension postgis;
CREATE EXTENSION

> select postgis_full_version();
                                                                               postgis_full_version                     
 POSTGIS="2.3.7 r16523" PGSQL="95" GEOS="3.6.2-CAPI-1.10.2 4d2925d" PROJ="Rel. 4.9.3, 15 August 2016" GDAL="GDAL 2.2.4, released 2018/03/19" LIBXML="2.7.8" LIBJSON="0.12" RASTER
(1 行记录)

5 配置远程访问

pg_hba.conf

> vim /home/postgres/95/data/pg_hba.conf
# TYPE  DATABASE        USER            ADDRESS                 METHOD

# IPv4 local connections:
host    all             all             127.0.0.1/32            md5
# IPv6 local connections:
host    all             all             ::1/128                 md5
# Allow replication connections from localhost, by a user with the
# replication privilege.

# add this line
host    all             all             0.0.0.0/0               md5

postgresql.conf

> vim /home/postgres/95/data/postgresql.conf

# change to this
listen_addresses='*'

6 重启

# Restart PostgreSQL or force the postmaster to reload the pg_hba.conf file by running
>  pg_ctl reload -D /home/postgres/95/data

7 卸载

uninstall-postgis-bundle-pg95x64-2.3.7-1.exe

uninstall-postgresql.exe
