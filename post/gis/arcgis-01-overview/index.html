介绍 ArcGIS 主要产品和模块的功能

2.2 常用组件

ArcGIS Dsktop 用来处理图像

ArcGIS Server 用于发布服务

3 Dsktop

ArcGIS Desktop 是基础部分，可供 GIS 专业人员创建、分析、管理和共享地理信息，以便决策者做出明智可靠的决策。 可用于创建地图、执行空间分析和管理数据。参考1

3.1 ArcCatalog

ArcCatalog 用来管理、访问和探究(Explore)存在的地理数据，无论数据存储的格式和地址（位于本地磁盘或者网络的其他位置）。它类似于Windows Explorer，但是为地理数据所设计的；使用ArcCatalog，可以改变数据的结构，如创建一个新的地理数据库(GeoDatabase)，装入现存的数据到地理数据库中，增加、删除属性表中的字段等。

3.2 ArcMap

ArcMap 用来完成所有基于地图的任务，包括地图制图、图编辑和分析等。使用ArcMap我们可以进行数据的浏览、符号化、查询、分析和输出等。ArcMap主要用于图形的编辑处理和符号化显示等。

3.3 ArcScene

主要进行一些三维显示和三维场景分析的模块。

4 ArcGIS Server

ArcGIS Server实现了一套标准的Web GIS服务（例如：制图，访问数据，地理编码等），支持企业级应用。ArcGISServer SDK使得开发人员可以建立集中式的GIS服务器来实现GIS功能，发布基于Web的GIS应用，执行分布式GIS运算等。
