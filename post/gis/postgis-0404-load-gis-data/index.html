PostGIS 加载数据|PostGIS Loading GIS Data

Posted by HostenWang on Monday, October 4, 2021

1 概述|Overview

PostGIS 是目前业界主流的地理数据库之一，提供如下空间信息服务功能：空间对象、空间索引、空间操作函数和空间操作符等。

本文介绍 #PostGIS 加载 GIS (矢量) 数据 的内容

2 步骤

  1. 调整坐标, 从 Xian_1980_3_Degree_GK_Zone_37 统一为 EPSG:4326

  2. 导入 PostGIS

  3. 从 GeoServer 读取 PostGIS 数据,坐标系选对

  4. CRS 选择从 data 计算

3 自动加载 shapefile 到 PostGIS 数据库1

3.1 自动导入数据

> shp2pgsql -c -D -s 4269 -i -I shape.shp myschema.roadstable > roads.sql 
> psql -d roadsdb -f roads.sql
